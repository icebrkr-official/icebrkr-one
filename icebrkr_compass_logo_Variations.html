Context & Brand

Design a unique animated logo for "icebrkr COMPASS" — a privacy-first AI Life OS product. Brand colors: Red #E8302A · Orange #F5A623 · Green #27AE60 · Dark #0F1923. Font: DM Serif Display (display), DM Sans (body), Space Mono (slogan). The logo must be trademark-unique — NOT a generic compass illustration.

Animation Specification

NEEDLE: Every 2–4 seconds, smoothly rotate to a new random bearing using CSS transition (cubic-bezier .25,.1,.1,1, 1.2s). Add brief 200ms overshoot, then settle. Randomise from: [0, 45, 90, 135, 180, 225, 270, 315] + random ±15° offset. Never reverse direction — always rotate clockwise. On click/tap → snap immediately to new random angle. SLOGAN TYPING: After 1s delay, type "Set your own Course with icebrkr" character by character (60ms/char). Use Web Audio API to generate a mechanical click tick (oscillator: square wave, freq 800Hz, duration 30ms, volume 0.04) on each character. After full type, show blinking cursor (orange, 0.8s blink). Loop typing animation every 12s. ENTRANCE: The entire compass SVG fades + scales from 0.9→1.0 over 600ms on load (ease-out).
